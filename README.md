# agentcore_line_chatbot
A sample LINE chatbot powered by AWS Lambda and AgentCore Runtime hosted Strabds Agents（Haiku 4.5）. It receives user messages via LINE webhook, invokes Bedrock to generate a reply, and sends it back via LINE Messaging API.
